# 🚀 Startup Success Predictor

An end-to-end Machine Learning project to predict the operational status of a startup (Operating, Acquired, Closed, or IPO) based on historical funding and business metrics.

## 📌 Project Goals
- Perform Exploratory Data Analysis (EDA) on the startup funding ecosystem.
- Build a predictive model to classify startup success.
- Deploy the model as an interactive web application.

## 🛠️ Tech Stack (Planned)
- **Data Manipulation:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn, XGBoost
- **Deployment:** Streamlit
